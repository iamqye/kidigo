# kidigo - Comments Repository
Alternatif repository comments system.


Repository ini digunakan khusus untuk menampung diskusi dan komentar dari Logbook [KIDIGO.MY.ID](https://www.kidigo.my.id) menggunakan **Giscus**.

### Deskripsi
* **Tujuan**: Menghubungkan interaksi pembaca logbook dengan GitHub Discussions.
* **Status**: Aktif.
* **Catatan**: Semua komentar yang diposting di logbook akan muncul di tab **Discussions** pada repository ini.


- Powered by [giscus](https://giscus.app)
- Website: [www.kidigo.my.id](https://www.kidigo.my.id)
